## Collection集合判空

1、springframework包下的CollectionUtils的isEmpty()
2、hutool包下的CollUtil的isEmpty()和isNotEmpty()
方法底层：
```java
public static boolean isEmpty(Collection<?> collection){
	return collection == null || collection.isEmpty();
}
```
