# 如何创建和切换分支？

# 题目详细答案
### 创建分支
使用`git branch`命令可以创建一个新的分支：

```plain
git branch <分支名>
git branch feature-jichi
```

### 切换分支
使用`git checkout`命令可以切换到另一个分支：

```plain
git checkout <分支名>
git checkout feature-jichi
```

### 创建并切换到新分支
你可以使用`git checkout -b`命令在创建新分支的同时切换到该分支：

```plain
git checkout -b <分支名>
git checkout -b feature-jichi
```
